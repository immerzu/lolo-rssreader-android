# GitHub Pages Anleitung

Diese Dateien sind fuer GitHub Pages vorbereitet:

- `docs/privacy.html`

So stellst du die Datenschutzseite online:

1. Erstelle auf GitHub ein neues Repository oder nutze ein bestehendes.
2. Lade den Ordner `docs` aus diesem Projekt in das Repository hoch.
3. Oeffne auf GitHub das Repository.
4. Gehe zu `Settings`.
5. Gehe links auf `Pages`.
6. Waehle als Quelle:
   - Branch: `main`
   - Folder: `/docs`
7. Speichern.
8. Warte kurz, bis GitHub Pages die Seite bereitstellt.

Typische URL:

- `https://DEIN-NAME.github.io/REPO-NAME/privacy.html`

Vorher noch anpassen:

1. In `docs/privacy.html` die Kontakt-E-Mail ersetzen.
2. Optional den Namen `Lolo` anpassen, falls du spaeter einen anderen Impressums- oder Kontakt-Namen verwenden willst.

Danach:

- genau diese URL in Google Play als Datenschutz-URL eintragen
